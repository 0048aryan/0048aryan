
===============================
        PERSONAL PORTFOLIO WEBSITE
===============================

PROJECT NAME:
Personal Portfolio / Resume Website

DESCRIPTION:
This is a responsive personal portfolio website built using
HTML and CSS. The website showcases personal information,
hobbies, skills, education, and contact details in a clean,
modern design.

The website is suitable for:
- Student portfolios
- Resume websites
- Personal branding
- Freelance showcase

--------------------------------------------------

TECHNOLOGIES USED:
- HTML5
- CSS3 (Flexbox, hover effects, responsive design)
- Video (MP4 format for hobby showcase)

--------------------------------------------------

FEATURES:
- Clean and modern UI
- Loveable-style dark gradient background
- Responsive layout (Desktop, Tablet, Mobile)
- Hover pop-up effects on videos and skill logos
- Education section with cards
- Contact section with call-to-action
- Optimized for GitHub Pages / Netlify hosting

HOW TO RUN LOCALLY:
1. Download or clone the project folder
2. Open the folder
3. Double-click on `index.html`
4. The website will open in your browser

--------------------------------------------------

HOW TO MAKE IT LIVE:
Option 1: GitHub Pages
- Upload all files to a GitHub repository
- Enable GitHub Pages in repository settings
- Access your site at: https://your-username.github.io

Option 2: Netlify
- Go to https://www.netlify.com
- Drag and drop the project folder
- Get an instant live URL

--------------------------------------------------

CUSTOMIZATION:
- Change name and text inside `index.html`
- Replace images with your own photos/logos
- Replace video files with your own MP4 videos
- Update email and social links in the contact section
